Degrees


Program to determine the shortest path between any two actors in the Hollywood film industry, based on the concept of "Six Degrees of Kevin Bacon". This involves constructing a representation of actors and their co-star relationships, and implementing a search algorithm to find the optimal connection.

Tic-Tac-Toe


AI opponent for the game of Tic-Tac-Toe that employs the Minimax algorithm to make optimal moves. The AI consistently wins or draws against any opponent.

Knights


Program to solve Knights and Knaves puzzles using propositional logic and model checking. The program accurately determines whether each character in a given puzzle is a knight or a knave based on the provided statements.

Minesweeper


AI capable of playing Minesweeper effectively. The AI utilizes logical reasoning to deduce safe and mined cells based on revealed numbers and apply probabilistic reasoning to make informed decisions in uncertain situations.

Page Rank


Python implementation of the PageRank algorithm to calculate the importance of web pages based on their link structure.

Heredity


Bayesian Network model to infer the probability distribution of gene copies and trait expression within a population based on observed data. Specifically, focus on a genetic trait influenced by multiple gene copies, such as hearing impairment linked to the GJB2 gene.

Crossword


Program to solve crossword puzzles by representing the problem as a constraint satisfaction problem (CSP) and employing backtracking search and other constraint propagation techniques.

Shopping


Nearest neighbor classifier to predict whether a user will make a purchase on an online shopping website based on given user behavior data.

Nim


AI agent that learns to play the game of Nim optimally through Q-learning. The agent effectively determines the best move in any given game state.

Traffic


TensorFlow-based neural network model to accurately classify traffic signs from the German Traffic Sign Recognition Benchmark (GTSRB) dataset.

Parser


A parser that can analyze English sentences using context-free grammar rules to determine sentence structure and extract noun phrases.

Attention


Utilized the BERT model to predict masked words in text and analyzed the attention mechanisms to understand how the model processes language.
